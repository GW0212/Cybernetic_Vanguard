게임 방법

TRON 느낌의 사이버 런너 게임입니다. 점프/슬라이드 없이 좌우 이동으로만 장애물을 피합니다.

  • 캐릭터는 자동으로 앞쪽(화면 안쪽)으로 달립니다.
  • PC: ← / → 키 또는 중앙의 ← 왼쪽 / 오른쪽 → 버튼으로 레인 이동
  • 모바일: 하단의 ← / → 버튼으로 좌우 이동
  • 장애물은 3개의 레인 중 랜덤으로 등장합니다. 같은 레인에서 가까이 오기 전에 다른 레인으로 이동해서 피하세요.
  • 코인은 장애물과 별도로 등장하며, 먹을수록 점수 배수가 조금씩 올라가 더 높은 점수를 노릴 수 있습니다.
  • 난이도(Easy / Normal / Hard)는 우측 상단 버튼에서 변경할 수 있습니다.
